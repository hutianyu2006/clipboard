蚌埠剪贴板FAQ

1.       这是什么？

答曰：这是一个被设计成类似于Ubuntu Paste的Web 3.0剪贴板。

2.       你这玩意和Ubuntu Paste有什么区别？

答曰：用户上传到Ubuntu Paste的数据将被存储在Ubuntu的服务器中，并且很难得知其是否使用了得当的加密算法。但是，蚌埠剪贴板基于IPFS，又对数据进行了AES-256-CTR加密，使您的数据尽可能得到妥善保存。

3.       你怎么想起来搞这个东西的？

答曰：那天我看见IPFS之后就有做这玩意的想法了，但苦于没有开放IPFS HTTP RPC API的IPFS公共网关而迟迟没有实现。后来，我发现了Infura IPFS Gateway的 IPFS HTTP RPC API 是开放的，于是便有了现在的蚌埠剪贴板。

4.       说了这么半天，IPFS是个啥子东西？

答曰：IPFS全称Interplanetary File System（星际文件系统），一个大区块链，详情请参考其百度百科。如果你不想研究它，那么你只要知道它可以比较安全的储存文件即可。

5.       那Web 3.0又是嘛？

答曰：这个一两句话讲不清楚，可以将百度百科和这篇知乎结合起来以便更好的理解它。

6.       我要这个剪贴板何用？

答曰：你可以拿它来存日记，存代码片段，存交易凭据，总之那些需要加密储存的稍隐私性的文件都可以，以后也会出加密上传文件的功能，这样就可以当一个无限存储容量的网盘用了——前提是你要找个地方把这些链接存起来，链接没了，数据就没了，数据无价！！！！！！

7.       我能为蚌埠剪贴板做些什么？

答曰：可以。这是一个开源的Github项目。您可以到Issues页面提交你发现的Bug或建议，但是我没时间处理Pull Requests，想对项目进行修改的把修改的代码放到Issues区，我会酌情应用。

8.       目前已知的需要改进的地方有哪些？
